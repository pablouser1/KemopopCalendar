# Kemopop! Calendar
Script to export birthdays to ics.

## Requirements
* Kemopop!
* Kemopop! Boys in Hawaii!
* Python3
* Decompiled project (you can use [GDRE](https://github.com/GDRETools/gdsdecomp))

## Installation
```bash
pip3 install -r requirements.txt
```

## Usage
```bash
python3 kemopop.py -h
```
