# Kemopop! Calendar
Script to export birthdays to ics.

## Requirements
* Kemopop! Boys in Hawaii!
* Python3
* Decompiled project (you can use [GDRE](https://github.com/GDRETools/gdsdecomp))

## Installation
```bash
pip3 install -r requirements.txt
```

## Usage
```bash
python3 kemopop.py 'PATH_TO_DECOMPILED_PROJECT' ['OUTPUT_PATH']
```

It should have generated a file named `kemopop.ics` in the script path.
